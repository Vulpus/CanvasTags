Welcome to Canvas Tags! This modular library is used to make objects on a canvas like buttons and inputs. It is designed to be as DOM-like a possible making it very easy to transfer HTML to your canvas. Useful features: 1. Main file only 1kb uncompressed(as of v1.0) 2. Easy to design and use modules 3. Github repo comes with prebuilt modules
